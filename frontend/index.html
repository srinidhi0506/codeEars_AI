function isPalindrome(str) {
  const clean = str.toLowerCase().replace(/[^a-z0-9]/g, '');
  return clean === clean.split('').reverse().join('');
}

Spoken explanation

This function checks whether a string reads the same forwards and backwards. It first cleans the input by lowercasing it and stripping out anything that isn't a letter or number, so punctuation and spacing don't get in the way. Then it compares the cleaned string to its own reverse — if they match, it's a palindrome.

Interview question

What happens with an empty string, and why does the regex strip step matter for inputs like "A man, a plan, a canal: Panama"?
